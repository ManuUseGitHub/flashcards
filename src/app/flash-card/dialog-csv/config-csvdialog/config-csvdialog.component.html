Our platform generates CSV files that are compatible with spreadsheet programs like Excel to add cards to our database. Some columns are required, but values can be inferred using grammar-based rules.

These rules use regular expressions (also known as regex) — a way to define patterns and recognition in text. While regex might seem technical and obscure, you don’t need to fully understand or master it. Helpful tools like ChatGPT can assist you in creating them without prior experience.

For instance, in French, words often require an article like “le” or “la.” A regex rule can detect if a translated word ends in “e” and automatically assign the article “la.” This removes the need for manual input and saves time.

Example rule: /e$/ → la means “if the word ends in 'e', use 'la'.”
